UserChoices::XmlConfigFileSource

Use an XML file as a source of choices. The XML file is parsed with XmlSimple('ForceArray' => false). That means that single elements like Mars are read as the value "Mars", whereas MarsVenus is read as ["Mars", "Venus"].
